11- Web Scrapping - Cours

Introduction

Qu’est ce que le web scraping ?

Le “web scraping” consiste à extraire de la donnée sur une language de balise html. Voici un emxemple de balise html:

<html>
  <head>
      <title> Un exemple de page HTML </title>
  </head>

  <body>
      <p>Un simple paragraphe</p>
  </body>
</html>

11.1- Récupérer des éléments d’une page

Une fois le code html télécharger, on va utiliser la librairie Python BeautifulSoup du package bs4 permettant d’analyser le contenu d’une page html.

  • Beautiful Soup est une bibliothèque Python pour extraire des données de fichiers HTML et XML. Il fonctionne avec votre analyseur favori pour fournir des moyens simplifiés de navigation, de recherche et de modification de l’arbre d’analyse. Il permet généralement aux programmeurs d’économiser des heures ou des jours de travail.
from bs4 import BeautifulSoup

response = requests.get("https://raw.githubusercontent.com/codelikerod/web-scraping/master/exemple1.html")
content = response.content

# on va analyser le contenu précédent avec BeautifulSoup 
parser = BeautifulSoup(content,'html.parser')
