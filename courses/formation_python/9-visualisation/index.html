9- La visualisation graphique avec Python - Cours

Objectifs

  • Comment générer des graphiques en Python avec matplotlib
  • Gerer plusieurs données dans un même graphique
  • Gérer les figures multi-graphiques
  • Créer des bar charts, histogrammes, courbes, nuages de points et boites à moustaches
  • Utiliser la librairie Seaborn

La bibliothèque Matplotlib est l’une des bibliothèques les plus utilisées pour tracer des données en Python.

De nombreux types de graphiques peuvent être développés avec cette bibliothèque: https://matplotlib.org/gallery/index.html

Pour illustrer certaines fonctionnalités du module Python matplotlib.pyplot (qui fournit un système de traçage similaire à celui de MATLAB), nous utiliserons une base de données de la station UQAM.

Matplotlib subplot

Lors de la création d’un graphique, matplotlib:

  • stocke un conteneur pour tous les graphiques
  • stocke un conteneur pour que le graphique soit positionné sur une grille
  • stocke les symboles visuels sur le graphique

Vous pouvez tracer différentes choses dans la même figure en utilisant la fonction de sous-tracé. Voici un exemple:

fig = plt.figure()
ax1 = fig.add_subplot(2,2,1) # up and left 
ax2 = fig.add_subplot(2,2,2) # up and right 
ax3 = fig.add_subplot(2,2,3) # down and left 
ax4 = fig.add_subplot(2,2,4) # down and right
plt.show()

Voici une documentation sur le sous-intrigue: https://matplotlib.org/api/pyplot_api.html#matplotlib.pyplot.subplot

4- Aller plus loin avec Seaborn

Pour compléter et enrichir nos visualiations, une nouvelle bibliothèque Python sera utilisée: Seaborn .

  • la bibliothèque Seaborn est basée sur matplotlib.
  • il permet de dessiner des graphiques plus complexes

Pour plus d’informations:

https://jakevdp.github.io/PythonDataScienceHandbook/04.14-visualization-with-seaborn.html

Ce lien présente une galerie de types de graphiques à réaliser avec Seaborn: https://seaborn.pydata.org/examples/index.html

https://seaborn.pydata.org/examples/index.html

import seaborn as sns
sns.distplot(data["Age"].dropna())
sns.set()
plt.show()


sns.kdeplot(data["Age"].dropna(), shade=True)
sns.set()
plt.xlabel("Age")
plt.show()


sns.lmplot(x="Age", y="Fare", hue="Pclass", data=data,fit_reg=False)
plt.show()


sns.jointplot("Age", "Fare", data=data, kind="reg")
plt.show()


sns.jointplot("Age", "Fare", data=data, kind="hex")
plt.show()


sns.boxplot(x=data["Age"])
plt.show()


sns.boxplot(x="Pclass", y="Age", data=data)
plt.show()


sns.violinplot(x="Pclass", y="Age", data=data)
plt.show()


fig, ax = plt.subplots()
fig.set_size_inches(12, 8)
sns.violinplot(x="Pclass", y="Age", data=data)
plt.show()
